Exercise One
  1. Frets 1-12 with no open strings
  2. Pick ONE natural note (no sharps or flats)
  3. Play notes in ascending order from the 6th (low E) string to the 1st (high E) string
  4. Once you reach the note on the 1st string reverse the order back up to the 6th string picking the same notes as you go
  5. If you can repeat the exercise 3 times without mistakes then you're ready to move on to the next note
  6. Now, pick the next note you want to learn and repeat the exercise TWICE
Exercise Two
  1. IF you've completed this exercise without making any mistakes then do it again except this time do it for the sharps and then the flats
  • * YES you can display the notes as you play
  • ** You do NOT need to memorise the location of the notes. Like learning to type, your fingers will remember their location the more you practice
Exercise One
  1. Similar to the Beginner exercise except this time pick TWO notes
  2. Ascend for the first note, and descend for the second note
  3. Perform this exercise for all natural notes TWICE
Exercise Two

Exercise Three
  1. Set the TIMER to a time you feel slightly uncomfortable with and perform any of the previous exercises, trying your best to get all the notes within that time
  2. Reduce the time and repeat
  • * YES you can display the notes as you play
  • ** You do NOT need to memorise the location of the notes. Like learning to type, your fingers will remember their location the more you practice
